# SQL Data Analysis & Aggregations Portfolio

## 📌 Project Overview
This project demonstrates practical SQL skills through a structured data analytics exercise focused on aggregations, filtering, grouping, and data analysis.

The project is based on a multi-table dataset including students, courses, enrollments, salaries, and projects. A total of 15 SQL queries were developed to extract insights and solve real-world business questions.
---
## 🎯 Aim of the Project
The goal of this project is to:
- Build strong SQL querying skills
- Understand relational data structures
- Apply aggregation and filtering techniques
- Extract meaningful insights from structured datasets
---
## 🧱 Dataset Description

### 1. Students
Contains student details including age and department.

### 2. Courses
Lists courses offered by departments with credit values.

### 3. Enrollments
Tracks which students are enrolled in which courses and their grades.

### 4. Salaries
Employee salary and bonus data across departments.

### 5. Projects
Company projects with associated departments and budgets.
---
## 🛠 Skills Demonstrated

- SQL Aggregations (SUM, AVG, COUNT, MIN, MAX)
- Filtering (WHERE, IN, BETWEEN, AND, OR)
- Grouping (GROUP BY, HAVING)
- Sorting & Limiting (ORDER BY, LIMIT)
- Data Analysis & Interpretation
